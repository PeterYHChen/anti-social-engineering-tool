COURANT INSTITUTE OF MATHEMATICAL SCIENCES

The Courant Institute of Mathematical Sciences is affiliated (but somewhat independent) of New York University. It teaches many things like pure math, applied math, statistics, and computer science.

Courant is ranked #1 for applied math in the world.
Courant is home to Facebook's AI Research Director, Yann LeCun.

Acceptance is based on skill and personal character. This project was made for a Practical Computer Security course, intended to be genuinely useful, although it could use some internal polishing around the edges. If it educates and provides a working extension, it has done well.
